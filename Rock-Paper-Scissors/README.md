# ✂️ Rock Paper Scissors Game

A Python implementation of the classic Rock Paper Scissors game.

Play against the computer and see who wins!

## Features

- Player vs Computer
- Random computer choices
- Score tracking
- Input validation
- Accepts uppercase and lowercase inputs
- Type "stop" anytime to quit

## How to Run

Run the following command:

```bash
python Rock_Paper_Scissors_Game.py
```

## Example

```text
Choose rock/paper/scissors:
rock

Computer chose paper
You chose rock

Lose!
Computer: 1 You: 0
```

## Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

## Concepts Practiced

- Lists
- Loops
- Conditional statements
- User input validation
- Random module
- Score management
