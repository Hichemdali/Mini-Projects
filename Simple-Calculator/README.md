# 🧮 Simple Calculator

A simple calculator written in Python that allows users to perform basic mathematical operations through an interactive menu.

## Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔢 Power
- 📊 Modulus
- ✅ Input validation
- 🚫 Division by zero protection
- 🔄 Continuous execution until the user chooses to exit

## How to Run

Make sure Python 3 is installed on your computer.

Run the following command:

```bash
python Simple_Calculator.py
```
