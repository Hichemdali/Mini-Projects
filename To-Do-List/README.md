# 📋 To-Do List Application

A simple command-line To-Do List application built with Python.

This project allows users to manage their daily tasks by adding, viewing, and removing tasks through an interactive menu.

## Features

- Add new tasks.
- Add multiple tasks at once.
- Display all tasks with automatic numbering.
- Remove tasks by selecting their number.
- Prevent removal when no tasks exist.
- Validate task numbers before deletion.
- Handle invalid menu choices.
- User-friendly interface.

## Menu

```text
===== TO-DO LIST =====

1 - Add Task
2 - Show Tasks
3 - Remove Task
4 - Exit
```

## Example

```text
1 - Study Python
2 - Go to the Gym
3 - Read a Book
```
